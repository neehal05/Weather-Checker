# 🌦️ Weather App using Flask and WeatherAPI

This is a simple weather application built using **Flask (Python)** and the **WeatherAPI.com** service.  
It allows users to enter any location and fetch current weather details like temperature, condition, and location name.

## 🔧 Features
- Get real-time weather data by location name
- Uses `WeatherAPI.com` for accurate weather info
- Simple and clean HTML interface
- Displays temperature, condition, and weather icon
